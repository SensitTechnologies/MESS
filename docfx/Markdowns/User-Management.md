## Creating Users
Users have 4 properties: 
* Email
* Username
* First name
* Last name

ID numbers are automatically generated when a new user template is created. Everything else must be entered manually by the technician creating the user.

## Editing Users
All of the properties of a user may be edited with the exception of the ID number. To do so, click the text boxes with the user's information and type in the changes. Then press the **save changes** button.

## Operators vs. Technicians
There are two roles that can be assigned to users. **Operators** are basic users that are only allowed to access the login and production log pages. **Technicians** are permitted to access administrative abilities.

## Deleting Users
Deleting a user deletes the information in the database relating to that user, which may compromise traceability purposes. As a result, it is recommended that you lock a user rather than deleting them if their access to the MESS software needs to be removed.

## Locking Users
Locking a user prevents them from logging in, functionally deleting them while preserving the data associated with their production logs. The user can be unlocked at any time to reactivate access to production logs.
